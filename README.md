# WGT Translator — 직독직해 워크북 메이커

더블유학원 교사용 자료 제작 툴. 영어 지문을 붙여넣으면 문맥순해(직독직해) 청크와 한국어 해석을 AI로 자동 생성하고, A4 인쇄용 워크북을 만들어 준다.

- 배포: GitHub Pages — https://onzzanglife-creator.github.io/WGT-Translator/
- 파일: `index.html` 단 하나 (로그인 없음, 주소 아는 사람만 사용)
- 관련 프로젝트: WGT-Program (학생용 학습 사이트, 별도 레포)

## 사용 흐름
1. 🔑 API 키 설정 — Gemini 키(`AIza...`, aistudio.google.com/apikey 무료 발급) 또는 OpenAI 키(`sk-...`, 크레딧 필요). 브라우저에만 저장됨.
2. ＋ 새 워크북 → 지문 붙여넣기 → ✨ 자동 분석
3. 청크/해석 편집 (합침·삭제·재분석, 원문↔청크 일치 자동 검증)
4. 🖨 인쇄 미리보기 → Ctrl+P로 종이/PDF 출력

## 워크북 구성 (A4 세로, 지문당)
머리글(더블유학원·학교/학년/이름/날짜 수기칸) + ①끊어읽기 ②해석쓰기(영→한) ③영작(한→영) ④어순배열(시드 셔플 — 재인쇄해도 동일) + 교사용 정답지(포함 선택)

## 데이터
- 작업물: localStorage `wgt_workbooks_v1` (브라우저별). 백업은 내보내기/가져오기(JSON).
- API 키: localStorage `wgt_wb_apikey`.

## AI 프롬프트 (index.html 내 buildPrompt)
문맥순해 원칙: 영어 어순 그대로, 청크 순서대로 읽어도 매끄러운 한국어. 연결어미·조사 필수, 긴 구조는 전치사구/관계사절/that절/접속사 경계에서 끊기, 문법 역할 판별(to부정사 목적어 "~하기로" — 병렬 오역 금지). 오역 패턴 발견 시 이 프롬프트에 정오답 예시를 추가하는 방식으로 개선한다.
- 모델: Gemini 2.5 Flash 우선, 404 시 2.0 Flash 폴백 / OpenAI는 gpt-4o-mini.

## 수정 시 원칙
- 파일 하나로 유지. 기존 코드 최소 수정. 프롬프트 개선은 예시 추가 방식.
